# Security Policy

## Reporting A Vulnerability

Please do not open a public issue for suspected vulnerabilities.

Email Oliver Hull with a description of the issue, affected versions, reproduction steps, and any relevant logs or proof of concept details. If the issue is confirmed, we will coordinate a fix and publish release notes with appropriate credit unless you prefer otherwise.

## Supported Versions

Open Memo is pre-1.0. Security fixes will target the latest released version unless a broader backport is explicitly noted.
